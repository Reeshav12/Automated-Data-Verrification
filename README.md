# Smart Verification

Smart Verification is now a React + Vite project with:

- a dedicated landing page
- a separate verification page
- an India flag inspired UI
- a DigiLocker-style verification experience

## Stack

- React 18
- Vite 5
- Custom CSS animations and 3D-style visual layout

## Local Run

```bash
npm install
npm run dev
```

Open:

```bash
http://127.0.0.1:5173/
```

## Production Build

```bash
npm run build
```

## Main Files

- `src/App.jsx`
- `src/styles.css`
- `src/main.jsx`
- `vite.config.js`

## Notes

- The old Streamlit/Python app files have been removed.
- This repo is now React-based.
